To run analysis_pcap_tcp.py, make sure you have installed the dpkt library in command prompt or in your ide

After running the file, it will prompt you to enter a file path for the pcap file that is to be tested, please enter a full file path (e.g. C:\Users\user\CSE310\assignment2.pcap) and make sure it's a .pcap file, else the program will prompt you to re-enter

After the program has started running, please wait for a few second for it to process then the outputs for each part will be printed out in a formatted form
